# 👨‍💻 Projetos - Curso Jovem Programador

Este repositório reúne os projetos desenvolvidos por **Carlos Lima** durante o curso **Jovem Programador**, oferecido pelo **Senac Santa Catarina**.

Cada projeto está organizado por disciplina/módulo, conforme os conteúdos abordados ao longo do curso.

## 📂 Módulos e Projetos

### 🔹 Lógica de Programação (`/logica-programacao`)

> Exercícios e projetos de introdução à lógica computacional com foco em algoritmos, estruturas de decisão e repetição.

- Calculadora simples em C#
- Conversor de temperaturas
- Validador de CPF

---

### 🔹 Banco de Dados (`/banco-de-dados`)

> Scripts SQL para criação de tabelas, consultas (DQL), manipulação de dados (DML) e modelagem de dados (MER/DER).

- Modelagem de vendas para papelaria
- Scripts Oracle e MySQL (DDL, DML, DQL)
- Consultas com junções e subconsultas

---

### 🔹 Desenvolvimento Web (`/desenvolvimento-web`)

> Projetos com HTML5, CSS3 e introdução ao JavaScript.

- Formulários de contato
- Página pessoal com layout responsivo
- Projeto "Montar Viagem"

---

### 🔹 Programação Orientada a Objetos (`/programacao-orientada-objetos`)

> Aplicações em C# com uso de classes, encapsulamento, listas e estruturas de dados.

- Sistema de notas de alunos
- Agenda de contatos
- Lista de tarefas com marcação de concluído

---

### 🔹 Projeto Integrador (`/projeto-integrador`)

> Projeto final integrando os conhecimentos de todo o curso.

- Sistema de Controle de Impressões:
  - Cadastro de usuários
  - Compra de pacotes de impressão
  - Registro de histórico e saldo
  - Interface gráfica em C# (Windows Forms)
  - Banco de dados relacional (MySQL)

---

## 📌 Sobre o Curso

O programa **Jovem Programador** é uma iniciativa do **Senac/SC** que prepara jovens para o mercado de TI, com foco em:

- Pensamento lógico
- Programação estruturada e orientada a objetos
- Desenvolvimento web
- Banco de dados
- Projetos reais integrados
